### High-Performance 5-Stage Pipelined Processor
[View on GitHub](https://github.com/Sankarsaan)
For this hardware project, I developed a 5-stage pipelined processor in Verilog. To keep things running fast, I engineered a custom synchronous hazard control unit to minimize stalls, and integrated a Branch Target Buffer (BTB) along with an LRU L1 cache to significantly reduce memory latency. I also validated the hardware accuracy by synthesizing it on an FPGA and successfully running C-based matrix multiplication and Fibonacci algorithms.

### Production-Grade Red-Black Tree
[View on GitHub](https://github.com/Sankarsaan)
I wanted to build a highly optimized, production-ready data structure, so I developed a Red-Black Tree in C++ that guarantees O(log n) operations. I utilized the Sentinel Node Pattern and recursive destructors to make absolutely sure there were zero memory leaks. When I ran it through a custom benchmarking suite I wrote, it actually reduced tree height by 99.7% compared to standard Binary Search Trees on sorted datasets!
